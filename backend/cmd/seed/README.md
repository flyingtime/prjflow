# 演示数据生成工具

这个工具用于生成完整的演示数据，方便展示系统的各种功能。

## 使用方法

### 基本用法

```bash
cd backend
go run cmd/seed/*.go
```

或者编译后运行：

```bash
cd backend
go build -o seed cmd/seed/*.go
./seed
```

### 重置数据库后生成数据

如果需要清空现有数据后重新生成：

```bash
./seed -reset
```

### 指定配置文件

```bash
./seed -config /path/to/config.yaml
```
